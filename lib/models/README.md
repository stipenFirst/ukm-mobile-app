# Models Flutter

Folder ini berisi class model data untuk memisahkan bentuk data dari tampilan UI.
UI lama masih dipertahankan agar fitur tidak rusak, tetapi model sudah siap dipakai jika ingin refactor bertahap dari `Map`/`List` menjadi object yang lebih rapi.

Model yang tersedia:
- `UserModel`
- `UkmModel`
- `PendaftaranModel`
- `AnggotaUkmModel`
- `NotifikasiModel`
- `SaranModel`
- `ApiResponseModel<T>`
